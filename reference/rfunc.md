# Generate random numbers from a [`DISTRIBUTION`](htinstatps://johnaponte.github.io/convdistr/reference/DISTRIBUTION.md) object

This is a generic method that calls the `rfunc` slot of the object

## Usage

``` r
rfunc(x, n)
```

## Arguments

- x:

  an object

- n:

  the number of random samples

## Value

a matrix with as many rows as `n` and as many columns as dimensions have
`distribution`

# Modify a the seed of a Distribution object

This create a new
[`DISTRIBUTION`](htinstatps://johnaponte.github.io/convdistr/reference/DISTRIBUTION.md)
object but with the specified seed

## Usage

``` r
set_seed(distribution, seed)
```

## Arguments

- distribution:

  a
  [`DISTRIBUTION`](htinstatps://johnaponte.github.io/convdistr/reference/DISTRIBUTION.md)
  object

- seed:

  the new seed

## Value

a
[`DISTRIBUTION`](htinstatps://johnaponte.github.io/convdistr/reference/DISTRIBUTION.md)
object of the same class

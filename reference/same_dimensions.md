# Check the dimensions of a list of distributions

Check the dimensions of a list of distributions

## Usage

``` r
same_dimensions(listdistr)
```

## Arguments

- listdistr:

  a list of
  [`DISTRIBUTION`](htinstatps://johnaponte.github.io/convdistr/reference/DISTRIBUTION.md)
  objects

## Value

return `TRUE` if all the dimensions are the same

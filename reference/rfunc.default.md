# Default function

Default function

## Usage

``` r
# Default S3 method
rfunc(x, n)
```

## Arguments

- x:

  an object of class different from
  [`DISTRIBUTION`](htinstatps://johnaponte.github.io/convdistr/reference/DISTRIBUTION.md)

- n:

  the number of random samples

## Value

No return value. Raise an error message.
